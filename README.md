## 🧪 Troubleshooting

### The app will not open

- Make sure the file finished downloading
- Check that you extracted the ZIP file if the download came in a ZIP
- Try running the file as administrator
- Download it again if the file looks incomplete

### Windows blocks the app

- Right-click the app file
- Select Properties
- If you see an Unblock option, select it
- Click Apply and try again

### The icon looks blurry

- Use a larger source image
- Start with a square image
- Avoid stretching the image before converting it
- Try a different image with sharper edges

### The output file is missing

- Check the output folder you chose
- Look in Downloads or Desktop if you are unsure
- Make sure the save step finished before closing the app
